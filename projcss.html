PROJECTS


Rock Paper Scissors

This is a basic Rock-Paper-Scissors game I made using Python, with no front end—just backend logic. The user plays through the terminal by entering their choice, and the computer randomly picks its move. It then compares both choices and announces the winner. It was a fun way to get comfortable with conditionals, loops, and working with Python's random module.

This is a backend-only weather app I made using Python. It takes a city name from the user through the terminal and fetches current weather data using an API.
